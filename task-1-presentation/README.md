# Task 1: Digital Literacy Awareness Infographic

## Tool Used
I created this infographic using canva, Using Canva is a fantastic choice because it completely removes the barrier between a great idea and a professional-looking design. Its intuitive drag-and-drop interface and massive library of templates mean you don't need a steep learning curve to create something visually stunning. It empowers you to bring your creative vision to life quickly, saving you valuable time while still delivering high-quality, polished results. Ultimately, it’s the perfect tool to ensure your visuals stand out and make a strong impact!

## Design Description
The infographic covers five key topics of digital literacy:

1. **What is Digital Literacy?** - Defines the concept as the ability to use digital technology responsibly
2. **Useful Digital Tools for Students** - Lists platforms like GitHub, LinkedIn, Google Workspace, and coding platforms
3. **Safe Internet Practices** - Covers password security, 2FA, phishing awareness, and software updates
4. **Professional Online Presence** - Guidelines for maintaining professional profiles
5. **Email Etiquette Essentials** - Key rules for professional email communication

The design uses a clean, modern layout with color-coded sections for easy navigation. Each section has an icon circle and clear headings.

## Reflection
The most interesting aspect of creating this infographic was balancing visual appeal with information density. I found it challenging to fit comprehensive information while keeping the design clean and readable. I learned that effective infographics require careful prioritization of content and strategic use of visual hierarchy to guide the reader's eye through the information.
